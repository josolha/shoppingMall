## 프로젝트 소개
이 프로젝트는 제가 개인적으로 향수에 대한 큰 관심을 바탕으로 시작되었습니다.  
특히, 제가 좋아하는 브랜드인 '조 말론'의 공식 웹사이트를 모티브로 삼아, 이를 모방하여 개인 프로젝트를 진행했습니다.
